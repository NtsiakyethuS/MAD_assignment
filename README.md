# Textbook Connect

Textbook Connect is an Android application designed to facilitate the buying and selling of textbooks within a campus community.

## Features

- **User Authentication**: Login and Registration screens for users to create accounts.
- **Home Market**: A searchable list of featured textbooks available for purchase.
- **Sell Textbooks**: Users can post their own textbooks for sale, including uploading a book picture and adding details like title, author, edition, and price.
- **User Profile**: A dedicated profile page where users can manage their personal information and profile picture.
- **Bottom Navigation**: Easy navigation between Home, Sell, and Profile sections.
- **Search Functionality**: Real-time search to find specific textbooks quickly.

## Technologies Used

- **Language**: Kotlin
- **UI Framework**: XML Layouts with Material Design components
- **Navigation**: BottomNavigationView
- **Image Handling**: Activity Result API for gallery image selection

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the application on an emulator or physical device.

## Future Enhancements

- Integration with a real database (e.g., Firebase or Room) for data persistence.
